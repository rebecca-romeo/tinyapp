# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (à la bit.ly).

!["screenshot of creating a tinyURL"](https://github.com/rebecca-romeo/tinyapp/blob/feature/basic-permission/docs/create-shortURL.png)

## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Start the project (using the `npm run` command).
- Run testing on the project (using the `npm run test` command).
